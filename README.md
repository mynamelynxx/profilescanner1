# ProfileScanner — Fabric Mod для Minecraft 1.21.4

Автоматически открывает `/profile <ник>` для каждого игрока из первых двух столбцов таб-листа.

## Что делает

- **K** — запускает сканирование
- **L** — останавливает в любой момент
- Для каждого игрока из первых 40 в табе (2 столбца × 20 строк):
  1. Отправляет `/profile <ник>`
  2. Ждёт открытия экрана (до 3 секунд)
  3. Держит экран открытым **1.5 секунды**
  4. Закрывает и переходит к следующему

## Сборка

### Требования
- **Java 21+** — [скачать](https://adoptium.net/)
- **Git** (опционально)

### Шаги

```bash
# 1. Войди в папку проекта
cd profilescanner

# 2. Запусти сборку (Gradle скачает всё сам, интернет нужен)
./gradlew build          # Linux / macOS
gradlew.bat build        # Windows

# 3. Готовый .jar будет в:
#    build/libs/profilescanner-1.0.0.jar
```

### Установка
1. Скопируй `profilescanner-1.0.0.jar` в папку `.minecraft/mods/`
2. Убедись что установлен **Fabric Loader 0.16+** и **Fabric API**
3. Запускай Minecraft 1.21.4

## Настройка клавиш

`Options → Controls → ProfileScanner` — можно переназначить K и L на любые клавиши.

## Как работает сортировка

Моды сортируют таб-лист **по алфавиту** (как делает сам Minecraft).  
Первые 20 = левый столбец, следующие 20 = правый столбец.

## Если `/profile` не открывает экран

Мод ждёт до 3 секунд. Если за это время экран не появился — пропускает игрока и идёт дальше. Это нормально для игроков в оффлайне или без профиля.
